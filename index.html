YouTube Frame Extractor

YouTube Frame Extractor is a Firefox extension that allows users to capture clean, high-quality frames directly from YouTube videos — without any UI elements such as:
- Video progress bar
- Video title
- Control buttons
- Comments or suggestions

The goal is to capture a perfectly clean image from the video, as if it was taken from the original production files.
